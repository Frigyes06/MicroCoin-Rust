# MicroCoin-Rust

MicroCoin node implemention in Rust. Heavily inspired by [MicroCoinCore](https://github.com/MicroCoinHU/MicroCoinCore)

# Running
You will need to have OpenSSL installed on your system.

Build and start the project with:
```
cargo run
```
